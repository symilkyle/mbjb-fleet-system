## ðŸ”§ Troubleshooting

### Problem: Page shows 404
**Solution:**
- Check GitHub Pages is enabled
- Verify branch is `main` and folder is `/`
- Wait 2-3 minutes after enabling
- Clear browser cache

### Problem: Page blank/white
**Solution:**
- Check browser console (F12) for errors
- Verify all files uploaded correctly
- Check `assets/` folder uploaded with all files

### Problem: Map not showing
**Solution:**
- Check internet connection (map needs internet)
- Wait few seconds for map tiles to load
- Refresh page (F5)

### Problem: 404 on navigation
**Solution:**
- âœ… Already fixed! Using HashRouter
- URLs will have `#` (e.g., `#/admin/tracking`)
- This is normal
